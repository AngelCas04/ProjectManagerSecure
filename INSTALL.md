# Instalacion de Project Manager Secure en Windows

Esta guia explica como descargar, instalar y abrir la aplicacion de escritorio.

## Descarga

Usa este enlace oficial del instalador:

[Descargar Project Manager Secure-Setup-0.1.0.exe](https://www.mediafire.com/file/roxlo4s97rvs99m/Project+Manager+Secure-Setup-0.1.0.exe/file)

## Requisitos

- Windows 10 u 11
- Conexion a internet para iniciar sesion y sincronizar el workspace

## Pasos de instalacion

1. Descarga el archivo `.exe` desde el enlace anterior.
2. Ejecuta `Project Manager Secure-Setup-0.1.0.exe`.
3. Si Windows muestra una advertencia por editor no reconocido, pulsa `Mas informacion` y luego `Ejecutar de todas formas`.
4. Espera a que termine la instalacion.
5. Abre `Project Manager Secure` desde el menu Inicio o desde el acceso directo del escritorio.

## Primer inicio

Al abrir la aplicacion:

1. Inicia sesion con tu cuenta.
2. Si aun no tienes una, usa la opcion `Crear cuenta`.
3. Si necesitas orientacion, en la pantalla de acceso hay un boton `¿Necesitas ayuda?`.

## Actualizaciones

La aplicacion esta preparada para actualizaciones del cliente. Si recibes una nueva version, instala la build mas reciente para mantener mejoras y correcciones.

## Solucion rapida de problemas

### La app no abre o se cierra sola

- Asegurate de usar la version mas reciente del instalador
- Cierra cualquier instancia previa antes de reinstalar
- Reinicia Windows si habia una version anterior abierta

### Windows muestra una advertencia de seguridad

Es normal mientras la build no tenga certificado de firma. Puedes continuar manualmente desde la ventana de Windows Defender SmartScreen.

### No puedo iniciar sesion

- Verifica tu correo y contrasena
- Asegurate de tener internet
- Si el problema continua, prueba primero en la version web
